## Fonctionnalities

* Temtem table on damages by type

* Temtem table on resistance by type

* Temtem datatable on every temtem with there types

## For Devs

* Start the project

    ```bash
    git clone https://github.com/TheSmartMonkey/temtem.git
    ```

    ```bash
    npm install
    ```

    ```bash
    npm install -g @angular/cli
    ```

    ```bash
    npm start
    ```
